# Agent Playground — Business Overview

## What This Is

You have built a **self-hosted AI agent management platform**. It lets users deploy a full AI automation
stack on their own server — with Claude-powered agents, task scheduling, chat, local LLMs (Ollama),
workflow automation (n8n), and a management dashboard — all from a single `docker compose up`.

The name: **Agent Playground** (also usable as "Agent Playground" for the SaaS landing page).

---

## The Core Business Idea

You are not selling software licenses. You are selling **outcomes delivered by AI agents running on
infrastructure you set up and manage for clients.**

Three revenue streams:

| Stream | What you do | Price range |
|---|---|---|
| **VPS Setup & Management** | Install the stack, configure domains/SSL, onboard client | $300–800 one-time + $99–299/mo retainer |
| **Web Development** | Build client websites (static or Next.js), host on their VPS with Nginx | $500–3000/site |
| **AI Automation Services** | Build n8n workflows + agent teams for specific client use cases | $500–2000/workflow + retainer |

These three naturally bundle: a client who needs a website also needs hosting, and a client with
hosting will eventually want AI automation. One sale leads to the next.

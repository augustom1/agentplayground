Friends report problems through your channel (Telegram / WhatsApp / in person). After the
release weekend, paste each report here using the template. One entry per problem, even if
rough — a screenshot description is enough.

## Report template

```
### [short title of the problem]
- Who: [friend's name]
- Date: [when it happened]
- What happened: [what they saw, copy their words]
- What they expected: [what they thought would happen]
- Screenshot: [attached / described / none]
- Machine: [Windows/Mac, anything unusual — low RAM, no Docker experience, etc.]
```
